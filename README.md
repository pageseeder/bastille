Bastille is Java library containing a collection of APIs and Berlioz generator implementations designed to provide all the basic functionalities for a Berlioz Website.

In particular, it provides:
- Connectors to PageSeeder
- Login and logout from PageSeeder via an external Website
- Indexing and search generators for Berlioz
- XML generators for Berlioz
- Web optimisation (minimizing, caching)
